## 数字精度丢失的典型问题

##### 1、两个浮点数相加 （部分浮点数相加导致精度丢失问题）

```
0.1 + 0.2 != 0.3 // true
```

##### 2、大整数运算

```
9999999999999999 == 10000000000000001 // true

16位数等于17位数？
```

##### 3、toFixed()

```
1.335.toFixed(2) (Chrome)
// 1.33

1.334.toFixed(2) (Chrome)
// 1.33

1.336.toFixed(2) (Chrome)
// 1.34

四舍五入，在五这个中间点有问题
```

## 数字精度丢失的解决方案

1、整数 运算结果不超过 Math.pow(2,53) 就不会丢失精度。

2、小数 把小数乘倍数，操作，再缩小回原来倍数。
